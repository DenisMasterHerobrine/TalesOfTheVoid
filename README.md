## Overview
"Tales of the Void" is a Hytale mod created during a 4-day ModJam event. The mod introduces a new dimension filled with unique biomes, structures, mobs, and lore, all centered around the theme of Void-Tech theme.

Implemented Custom Content:
- Music, sound effects (see Credits section)
- Mobs
- Animations
- Dimension
- Day-and-night atmosphere
- Structures
- Biomes
- Blocks
- Block overlays
- Loot
- Items
- Decorative elements
- Dungeon mechanics

## How to Start?

It's easy to start. Just craft the Abyss Portal in the **Arcane Table** and go through it to access the dimension. Explore, survive and come back... if you can.

## Development Notes
Hytale is still in an early alpha stage, which introduced a significant number of limitations and unexpected challenges ðŸ¥€

###  Some examples:
- Attack animations are partially hardcoded
(or at least that's how we understood it - figuring it out would have taken a lot of time, so we decided to work around it and adapt the custom mob models to the existing attacks of some mobs that are already in the game)

- Structure generation caused numerous edge cases and instability

Portal logic required multiple experimental implementations before working reliably, several core systems were simply unavailable
(Huge thanks to joshieman for the opportunity to use [Shared Structures](https://www.curseforge.com/hytale/mods/shared-structures)!
However, in the final version we decided to drop the library. We changed our approach due to Hytale breaking changes introduced in the last patch)
